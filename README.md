# Customer Churn Prediction System

## Deskripsi Project
Project ini dibuat untuk memenuhi tugas UAS Data Mining.

Sistem menggunakan algoritma Random Forest
untuk melakukan prediksi customer churn
berdasarkan dataset Telco Customer Churn.

---

## Features
- Home Dashboard
- Dataset Overview
- Customer Churn Prediction
- Model Evaluation
- Visualization Dashboard
- Deployment Streamlit Cloud
